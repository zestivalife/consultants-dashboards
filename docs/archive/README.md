# Documentation Archive

This directory stores documents that are no longer authoritative.

## Structure

- `duplicates/`: duplicate or superseded copies retained for traceability.
- `historical/`: historical implementation notes and retired operational notes.
- `temporary/`: temporary documents awaiting classification.

Archived documents must not be used as implementation source of truth unless explicitly promoted back into an active documentation category.
