# Documentation Status Legend

| Status | Meaning |
|---|---|
| DRAFT | Work in progress; not implementation authority. |
| REVIEW | Ready for review; not final authority. |
| APPROVED | Accepted and may guide implementation. |
| ACTIVE | Current operating source of truth. |
| SUPERSEDED | Replaced by a newer document. |
| ARCHIVED | Historical record; retained for traceability only. |

## Lifecycle Rule

Only `APPROVED` and `ACTIVE` documents may be used as implementation authority.

`SUPERSEDED` and `ARCHIVED` documents must not guide implementation unless explicitly referenced for history.
