## Classification Rules

- Governance documents define how work is performed.
- Architecture documents define durable system structure.
- Platform capability documents define reusable shared capabilities.
- Domain documents define durable business domains.
- Product documents define product-specific behavior.
- Service documents define deployable service behavior.
- Operations documents define how the platform runs.
- Delivery documents define roadmap, release, testing, and acceptance.
- Milestone documents define temporary execution scope.
- ADRs record approved architecture decisions.
- Archive documents are retained but not authoritative.
