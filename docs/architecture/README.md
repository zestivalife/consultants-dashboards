# Architecture Documentation

This directory contains durable architecture decisions and specifications. Architecture documents are source-of-truth references and should not contain temporary milestone execution notes.

## Structure

- `enterprise/`: enterprise-wide architecture, workspace architecture, application information architecture, and design system architecture.
- `domains/`: durable business and technical domain models.
- `api/`: API standards and shared contracts.
- `data/`: database standards, schema principles, migration rules, and data architecture.
- `security/`: security architecture, policies, and secure implementation standards.
